# 7th JoJo Personality Test Site

A Game Boy-style recreation of the personality evaluation from *The 7th Stand User*.

## Play

Open `index.html` in a browser.

## Links

- Project: https://github.com/mncrftfrcnm/7th_jojo_personality_test_site
- Original game downloads: https://7thjojo.fandom.com/wiki/FAQ#Download_links

## Survey builder

Create a survey file:

```bash
python gameboy_survey_builder.py wizard survey.json
```

Build it:

```bash
python gameboy_survey_builder.py build survey.json -o index.html
```

Check a survey file:

```bash
python gameboy_survey_builder.py validate survey.json
```
